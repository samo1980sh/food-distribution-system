# مصفوفة الأدوار والصلاحيات — RBAC Foundation

## الأدوار

| الدور | الاستخدام |
|---|---|
| `super_admin` | إدارة النظام بالكامل، ويتجاوز Gates عندما يكون الحساب فعّالًا. |
| `manager` | إدارة شاملة للعمليات والتقارير، دون رؤية أو تعديل حسابات Super Admin. |
| `supervisor` | الإشراف التشغيلي والاعتماد والتقارير التشغيلية والمالية المسموحة. |
| `warehouse_keeper` | المستودعات والمخزون وحركات المخزون وتحميل السيارات وتقارير المخزون والصلاحية. |
| `accountant` | التحصيلات والذمم والإغلاقات والتقارير المالية. |
| `sales_representative` | صلاحيات تشغيلية أولية للمندوب، مخصصة لاحقًا لـ API/Flutter ولا تسمح بدخول لوحة الإدارة حاليًا. |
| `driver` | صلاحيات تشغيلية أولية للسائق، مخصصة لاحقًا لـ API/Flutter ولا تسمح بدخول لوحة الإدارة حاليًا. |

## مبادئ التوزيع

- الصلاحية مقسمة إلى عرض وإنشاء وتعديل وحذف واعتماد وإلغاء وطباعة، بدل منح قسم كامل بصلاحية عامة.
- إجراءات المستندات مرتبطة بحالة السجل؛ مثلًا لا يمكن تعديل فاتورة بعد تأكيدها.
- الحساب غير الفعّال يُرفض عبر Gate العام حتى لو كان Super Admin.
- آخر Super Admin فعّال محمي من التعطيل أو تغيير الدور من شاشة المستخدمين.
- المدير لا يرى حسابات Super Admin في جدول المستخدمين، ولا يظهر له هذا الدور ضمن خيارات الإسناد.
- السائق والمندوب لا يملكان `admin.access` في هذه المرحلة.

## مجموعات الصلاحيات

- دخول لوحة الإدارة والـDashboard.
- المستخدمون وإسناد الأدوار.
- المناطق والموظفون والعملاء.
- الوحدات والتصنيفات والمنتجات.
- خطوط التوزيع والسيارات والمستودعات.
- أرصدة وحركات المخزون.
- أوامر تحميل السيارات واعتمادها وإلغاؤها وطباعتها.
- فواتير المبيعات وتأكيدها وإلغاؤها وطباعتها.
- مرتجعات المبيعات وتأكيدها وإلغاؤها وطباعتها.
- دفعات العملاء وتأكيدها وإلغاؤها وطباعتها.
- مصاريف السيارات واعتمادها ورفضها وطباعتها.
- الإغلاقات اليومية وتحديث الإجماليات وتأكيدها وإلغاؤها وطباعتها.
- جميع التقارير الحالية كلٌ بصلاحية مستقلة.

## المرحلة التالية: Data Scopes

هذه المصفوفة تحدد **ماذا يستطيع المستخدم أن يفعل**. الحزمة التالية ستحدد **على أي سجلات يستطيع تنفيذ ذلك**، مثل:

- مناطق المشرف وخطوطه.
- مستودعات أمين المستودع.
- سيارة السائق.
- خط المندوب وعملاؤه.
- تقييد Dashboard والتقارير والطباعة والاستعلامات والـAPI بالنطاق نفسه.
